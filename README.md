# Weather Dashboard

Aplikasi Weather Dashboard sederhana untuk tugas akhir praktikum:
Fitur:
- Current weather: temperature, humidity, wind, icon, timestamp
- 5-day forecast (min/max + icon + description)
- Search dengan autocomplete (OpenWeatherMap Geocoding)
- Save favorite cities (localStorage)
- Toggle Celsius/Fahrenheit
- Dark / Light theme toggle
- Manual refresh & auto-update setiap 5 menit
- Loading indicators

## Running
Opsi 1 — Live Server (VSCode)
1. Klik kanan index.html
2. Pilih Open with Live Server
3. Buka di browser: http://localhost:5500

Opsi 2 — Python HTTP Server
1. Buka terminal di folder project:
2. python -m http.server 5500
3. Lalu buka: http://localhost:5500

## Setup
1. Daftar di https://openweathermap.org
2. Ambil API Key
3. Buka app.js
4. Ganti dengan API key Anda
